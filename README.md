# Ajazz 日本官网 - 最简版（方案 C）

## 项目信息
- **项目名**：ajazz-japan-landing
- **目标**：快速上线验证日本市场需求
- **上线时间**：3-5 天
- **技术栈**：Next.js + Vercel（免费）

### 1. Hero 区域
- 品牌 Logo
- 主标语（日语）
- CTA 按钮（查看产品）

### 2. 产品列表（3-5 款核心产品）
- AK029（猫键盘）
- AK820 Pro
- AK980
- AJ159 系列鼠标

每款产品卡片：
- 产品图
- 产品名（日语）
- 简短描述
- 价格（日元）
- 购买按钮（跳转亚马逊日本/乐天）

### 3. 品牌优势
- 免费配送
- 1 年保修
- 客服支持

### 4. 用户评价（3-4 条）
- 从 ajazzstore.com 翻译日语

### 5. 页脚
- 联系邮箱
- 社交媒体链接（X @AjazzJapan）
- 政策链接（配送/条款/联系）
